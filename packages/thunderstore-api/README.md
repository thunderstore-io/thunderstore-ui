# thunderstore-api

Thunderstore API client

## Scripts

- `pnpm run build`: Builds the project
- `pnpm run dev`: Builds the project & watches files for changes, triggering a rebuild
- `pnpm run test`: Runs the test suite once
- `pnpm run watch`: Runs the test suite & watches files for changes, triggering a rerun

Note that when running scripts from the monorepo's root, you need to use
e.g. `pnpm --filter @thunderstore/thunderstore-api run build`.
