# @thunderstore/ts-api-react

React integration for the Thunderstore API. Provides hooks and context to easily consume API data in React components.

## Features

-   **Hooks**: Custom hooks for fetching data (e.g., `usePackage`, `useCommunity`).
-   **Context**: Provides API configuration and state to the component tree.
-   **Integration**: Bridges `@thunderstore/thunderstore-api` with React's lifecycle.

## Scripts

- `yarn run build`: Builds the project
- `yarn run dev`: Builds the project & watches files for changes, triggering a rebuild
