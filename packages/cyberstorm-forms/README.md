# @thunderstore/cyberstorm-forms

A collection of form components and hooks built for the Cyberstorm design system.

## Features

-   **Form Components**: Reusable inputs, selects, and other form elements styled with Cyberstorm.
-   **Validation**: Integration with `zod` for schema validation.
-   **State Management**: Built on top of `react-hook-form`.
-   **Actions**: Helpers for handling form submissions and API interactions.

## Usage

```tsx
import { Form, TextInput } from "@thunderstore/cyberstorm-forms";
import { useForm } from "react-hook-form";

// ...
```

## Scripts

- `yarn run build`: Builds the project
- `yarn run dev`: Builds the project & watches files for changes, triggering a rebuild
